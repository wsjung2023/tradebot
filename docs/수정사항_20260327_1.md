## 4. 원인 요약

### 원인 1 — Legacy Fallback (직접 원인, 반드시 수정)

`user-kiwoom.service.ts` 158~163번 줄:

에이전트가 에러를 반환할 때 `AgentTimeoutError`가 아닌 일반 `Error`로 전달되므로,
서버의 catch 블록이 Replit 서버 자체에서 키움 WebSocket 직접 연결(Legacy)을 시도한다.

Replit 서버는 유동 IP를 가지므로 키움 WebSocket에 직접 연결하면 실패한다.
(아키텍처 설계 원칙: "키움 API는 항상 집 PC 에이전트를 통해서만 호출"에도 위반)

### 원인 2 — Python WebSocket 헤더 형식 (간접 원인, 함께 수정)

`kiwoom-agent.py` 189~192번 줄:

Python `websocket-client` 라이브러리는 `header={}` 딕셔너리 형식으로 전달 시
실제 HTTP Upgrade 요청 헤더에 포함되지 않는 버전이 존재한다.
리스트 형식(`["key: value"]`)이 표준이며 확실하게 전달된다.
이것이 원인 1을 촉발시키는 근본 원인이다.

## 5. 수정 지시 — 파일 2개, 수정 순서 엄수

### [수정 1] `server/services/user-kiwoom.service.ts` — 154~164번 줄

**이 수정만 해도 브라우저 에러 100013은 즉시 사라진다.**

**현재 코드 (154~164번 줄)**:
```typescript
  async getConditionList(userId: string) {
    try {
      const response = await callViaAgent(userId, "condition.list", {});
      return response?.output ?? response ?? [];
    } catch (error) {
      if (error instanceof AgentTimeoutError) throw error;
      const legacyKiwoom = await this.getLegacyServiceForUser(userId);
      const response = await legacyKiwoom.getConditionList();
      return response?.output ?? response ?? [];
    }
  }
```

**수정 후 코드 (154~157번 줄로 축소)**:
```typescript
  async getConditionList(userId: string) {
    const response = await callViaAgent(userId, "condition.list", {});
    return response?.output ?? response ?? [];
  }
```

변경 내용:
- try-catch 블록 전체 제거
- legacy fallback 3줄 (`getLegacyServiceForUser`, `legacyKiwoom.getConditionList()`, 반환) 제거
- `callViaAgent` 에러는 그대로 상위로 전파 → 클라이언트에 정확한 에이전트 에러 메시지 표시
- `AgentTimeoutError` import 구문은 다른 함수에서 사용 중이므로 유지 (제거하지 말 것)

---

### [수정 2] `agent/kiwoom-agent.py` — 189~192번 줄

**이 수정으로 에이전트 WebSocket 인증이 성공하여 근본 원인까지 해결된다.**

**현재 코드 (187~196번 줄)**:
```python
    ws_app = websocket.WebSocketApp(
        ws_url,
        header={
            "api-id": api_id,
            "Authorization": f"Bearer {token}",
        },
        on_open=on_open,
        on_message=on_message,
        on_error=on_error,
    )
```

**수정 후 코드 (187~196번 줄)**:
```python
    ws_app = websocket.WebSocketApp(
        ws_url,
        header=[
            f"api-id: {api_id}",
            f"Authorization: Bearer {token}",
        ],
        on_open=on_open,
        on_message=on_message,
        on_error=on_error,
    )
```

변경 내용:

| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| `header` 타입 | `dict` (`{}`) | `list` (`[]`) |
| `api-id` 형식 | `"api-id": api_id` | `f"api-id: {api_id}"` |
| `Authorization` 형식 | `"Authorization": f"Bearer {token}"` | `f"Authorization: Bearer {token}"` |
| 변경 줄 수 | — | 189~192번 줄 (4줄 교체) |
