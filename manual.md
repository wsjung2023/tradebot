## 1. 회원가입하기

### 방법 1: 이메일로 가입하기

1. 앱을 열면 로그인 화면이 나와요
2. 화면 아래에 있는 **"회원가입"** 글자를 클릭해요
3. 빈칸을 채워요:
   - **이름**: 내 이름을 써요 (예: 홍길동)
   - **이메일**: 내 이메일 주소를 써요 (예: hong@gmail.com)
   - **비밀번호**: 8자 이상으로 만들어요 (예: mypassword123)
4. **"회원가입"** 버튼을 눌러요
5. 끝! 자동으로 로그인돼요

### 방법 2: 구글로 가입하기 (더 쉬워요!)

1. 앱을 열면 로그인 화면이 나와요
2. **"Google 로그인"** 버튼을 눌러요
3. 내 구글 계정을 선택해요
4. 끝! 바로 사용할 수 있어요

## 2. 로그인하기

이미 가입했다면 로그인만 하면 돼요.

1. 앱을 열어요
2. **이메일**과 **비밀번호**를 입력해요
3. **"로그인"** 버튼을 눌러요
4. 대시보드 화면이 나오면 성공!

> **팁**: 구글로 가입했다면 **"Google 로그인"** 버튼을 눌러요

---

## 3. 계좌 연결하기

주식을 사고팔려면 먼저 계좌를 연결해야 해요.

### 모의투자 계좌 (연습용 - 추천!)

처음이라면 모의투자로 연습해보세요. 진짜 돈이 들지 않아요!

1. 왼쪽 메뉴에서 **"설정"**을 눌러요
2. **"계좌 관리"** 부분을 찾아요
3. **"계좌 추가"** 버튼을 눌러요
4. 빈칸을 채워요:
   - **계좌 이름**: 아무 이름이나 써요 (예: 연습용계좌)
   - **계좌 번호**: 키움증권에서 받은 번호를 써요
   - **계좌 유형**: **"모의투자"**를 선택해요
5. **"저장"** 버튼을 눌러요

### 실전 계좌 (진짜 돈으로 투자)

> **주의**: 실전 계좌는 진짜 돈이 들어가요. 신중하게 사용하세요!

1. 왼쪽 메뉴에서 **"설정"**을 눌러요
2. **"계좌 관리"** 부분을 찾아요
3. **"계좌 추가"** 버튼을 눌러요
4. 빈칸을 채워요:
   - **계좌 이름**: 구분하기 쉬운 이름 (예: 실전투자계좌)
   - **계좌 번호**: 키움증권 실전 계좌 번호
   - **계좌 유형**: **"실전"**을 선택해요
   - **API 키**: 키움증권에서 발급받은 앱키
   - **API 시크릿**: 키움증권에서 발급받은 앱시크릿
5. **"저장"** 버튼을 눌러요

## 4. 대시보드 살펴보기

로그인하면 가장 먼저 보이는 화면이에요.

### 화면 구성

```
┌─────────────────────────────────────────────────┐
│  왼쪽 메뉴        │      메인 화면              │
│                   │                             │
│  - 대시보드       │   내 자산 현황              │
│  - 거래           │   ┌─────────────────────┐   │
│  - AI 분석        │   │  총 자산: 1,000만원  │   │
│  - 자동매매       │   │  수익률: +5.2%      │   │
│  - 포트폴리오     │   └─────────────────────┘   │
│  - 거래 내역      │                             │
│  - 관심종목       │   보유 주식 목록            │
│  - 설정           │   - 삼성전자 10주           │
│                   │   - 카카오 5주              │
└─────────────────────────────────────────────────┘
```

### 주요 정보

- **총 자산**: 내가 가진 돈 + 주식 가치의 합계
- **수익률**: 처음 투자한 돈 대비 얼마나 벌었는지
- **보유 주식**: 내가 지금 가지고 있는 주식들
- **자산 추이 그래프**: 30일 동안 내 자산이 어떻게 변했는지 보여줘요

## 5. 자동매매 설정하기

가장 중요한 기능이에요! 설정해두면 AI가 알아서 주식을 사고팔아요.

### 자동매매 켜기

1. 왼쪽 메뉴에서 **"자동매매"**를 눌러요
2. **"자동매매 활성화"** 스위치를 켜요 (초록색이 되면 켜진 거예요)
3. 끝! 이제 AI가 알아서 투자해요

### AI 모델 선택하기

여러 종류의 AI 중에서 고를 수 있어요.

| AI 이름 | 특징 | 추천 대상 |
|---------|------|----------|
| gpt-5.1 | 가장 똑똑해요 | 모든 사람 (기본값) |
| gpt-5.1-chat-latest | 빠르게 분석해요 | 급한 결정이 필요할 때 |
| gpt-5-mini | 저렴해요 | 비용을 아끼고 싶을 때 |

### 매매 금액 설정하기

1. **"1회 최대 투자금"**을 정해요 (예: 100만원)
   - 한 번에 이 금액 이상은 투자하지 않아요
2. **"총 투자 한도"**를 정해요 (예: 1000만원)
   - 전체 투자금이 이 금액을 넘지 않아요

### 자동매매가 작동하는 시간

- **평일 아침 9시 ~ 오후 3시 30분** (주식시장 열리는 시간)
- 주말과 공휴일에는 쉬어요
- 컴퓨터를 꺼도 서버에서 계속 돌아가요!

## 6. 수동으로 주식 사고팔기

AI 대신 직접 주식을 사고 싶을 때 사용해요.

### 주식 사기 (매수)

1. 왼쪽 메뉴에서 **"거래"**를 눌러요
2. 위에 있는 검색창에 주식 이름이나 코드를 써요
   - 예: "삼성전자" 또는 "005930"
3. 차트와 정보가 나타나요
4. 아래 주문 창에서:
   - **"매수"** 탭을 선택해요
   - **주문 유형**: 
     - "지정가" = 내가 원하는 가격에 사기
     - "시장가" = 지금 바로 사기 (가격은 시장이 정해요)
   - **가격**: 지정가일 때만 써요 (예: 75000)
   - **수량**: 몇 주 살지 써요 (예: 10)
5. **"매수"** 버튼을 눌러요
6. 주문 완료!

### 주식 팔기 (매도)

1. 왼쪽 메뉴에서 **"거래"**를 눌러요
2. 검색창에 팔고 싶은 주식을 검색해요
3. 아래 주문 창에서:
   - **"매도"** 탭을 선택해요
   - **주문 유형**: 지정가 또는 시장가
   - **가격**: 지정가일 때만 써요
   - **수량**: 몇 주 팔지 써요
4. **"매도"** 버튼을 눌러요
5. 주문 완료!

> **팁**: 시장가로 주문하면 바로 체결되지만, 가격은 시장 상황에 따라 달라져요

## 7. AI 분석 보기

AI가 주식을 어떻게 분석했는지 볼 수 있어요.

### AI 분석 화면 열기

1. 왼쪽 메뉴에서 **"AI 분석"**을 눌러요
2. 분석하고 싶은 주식 코드를 입력해요 (예: 005930)
3. **"분석 시작"** 버튼을 눌러요
4. AI가 열심히 분석해요 (조금 기다려요)
5. 결과가 나와요!

### AI 분석 결과 읽는 법

```
┌─────────────────────────────────────────────────┐
│  삼성전자 (005930) AI 분석 결과                 │
├─────────────────────────────────────────────────┤
│  추천: 매수 ⭐⭐⭐⭐                            │
│  신뢰도: 85%                                    │
│                                                 │
│  분석 내용:                                     │
│  - 재무 상태가 좋아요                           │
│  - 최근 실적이 좋았어요                         │
│  - 외국인/기관이 많이 사고 있어요               │
│                                                 │
│  레인보우 차트 위치: 5번 라인 (매수 구간)       │
└─────────────────────────────────────────────────┘
```

### 추천 등급 설명

| 등급 | 의미 | AI의 생각 |
|------|------|----------|
| 강력 매수 | 지금 바로 사세요! | 정말 좋은 기회예요 |
| 매수 | 사는 게 좋아요 | 괜찮은 기회예요 |
| 중립 | 기다려 보세요 | 아직 판단이 어려워요 |
| 매도 | 파는 게 좋아요 | 조심해야 해요 |
| 강력 매도 | 지금 바로 파세요! | 위험할 수 있어요 |

## 8. 관심종목 등록하기

자주 보고 싶은 주식을 저장해둘 수 있어요.

### 관심종목 추가하기

1. 왼쪽 메뉴에서 **"관심종목"**을 눌러요
2. **"종목 추가"** 버튼을 눌러요
3. 주식 코드를 입력해요 (예: 005930)
4. **"추가"** 버튼을 눌러요
5. 이제 목록에서 빠르게 볼 수 있어요!

### 가격 알림 설정하기

원하는 가격이 되면 알려줘요.

1. 관심종목 목록에서 주식을 선택해요
2. **"알림 설정"** 버튼을 눌러요
3. 알림 받고 싶은 가격을 입력해요
   - 예: 삼성전자가 80,000원 되면 알림
4. **"저장"** 버튼을 눌러요

## 9. 거래 내역 확인하기

내가 사고판 기록을 볼 수 있어요.

### 거래 내역 보기

1. 왼쪽 메뉴에서 **"거래 내역"**을 눌러요
2. 모든 거래 기록이 나와요

### 거래 기록 읽는 법

```
┌──────────────────────────────────────────────────────────┐
│ 날짜       │ 종목     │ 유형 │ 가격    │ 수량 │ 상태    │
├──────────────────────────────────────────────────────────┤
│ 2025-11-28 │ 삼성전자 │ 매수 │ 75,000  │ 10   │ 체결    │
│ 2025-11-27 │ 카카오   │ 매도 │ 52,000  │ 5    │ 체결    │
│ 2025-11-26 │ 네이버   │ 매수 │ 180,000 │ 2    │ 취소    │
└──────────────────────────────────────────────────────────┘
```

### 상태 설명

| 상태 | 의미 |
|------|------|
| 대기 | 아직 거래가 안 됐어요 |
| 체결 | 거래가 완료됐어요 |
| 취소 | 주문을 취소했어요 |
| 실패 | 문제가 생겨서 안 됐어요 |

## 11. 자주 묻는 질문

### Q: 자동매매를 켜두면 컴퓨터를 꺼도 되나요?
**A: 네!** 자동매매는 서버에서 돌아가기 때문에 컴퓨터를 꺼도, 브라우저를 닫아도 계속 작동해요.

### Q: 주말에도 자동매매가 되나요?
**A: 아니요.** 주식시장이 열리는 평일 오전 9시 ~ 오후 3시 30분에만 작동해요.

### Q: 모의투자로 연습하면 진짜 돈이 드나요?
**A: 아니요!** 모의투자는 가짜 돈으로 연습하는 거예요. 안심하고 연습하세요.

### Q: 얼마나 벌 수 있나요?
**A:** 투자는 항상 위험이 있어요. AI가 도와주지만 100% 수익을 보장할 수는 없어요. 여유 자금으로 투자하세요.

### Q: AI가 잘못된 결정을 하면 어떻게 하나요?
**A:** 자동매매 설정에서 **"손절 기준"**을 정해두면, 너무 많이 잃기 전에 자동으로 팔아요. 예를 들어 -5%가 되면 자동으로 손절.

### Q: 비밀번호를 잊어버렸어요
**A:** 로그인 화면에서 **"비밀번호 찾기"**를 눌러서 이메일로 재설정할 수 있어요.

### Q: 앱이 안 될 때는요?
**A:** 
1. 인터넷 연결을 확인해요
2. 페이지를 새로고침해요 (F5 키)
3. 다시 로그인해요
4. 그래도 안 되면 잠시 후 다시 시도해요
